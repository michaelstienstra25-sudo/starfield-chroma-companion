Recommended installer for most users.

Run StarfieldChromaCompanionSetup-v0.1.3-alpha.exe.

The setup assistant:
- Searches Steam libraries and common install paths for Starfield.
- Lets you browse manually if Starfield is installed elsewhere.
- Checks that sfse_loader.exe exists in the selected Starfield folder.
- Checks that Node.js is installed.
- Installs the SFSE plugin DLLs to Data\SFSE\Plugins.
- Installs the companion launcher app to %LOCALAPPDATA%\StarfieldChromaCompanion.
- Saves the selected Starfield folder in the config.
- Lets you choose Desktop and Start Menu shortcuts.

After installing, open Razer Chroma and make sure CHROMA APPS is enabled. If Chroma Apps is disabled, the keyboard may stay on Spectrum Cycling even though the SDK connection succeeds.

This is a single-file self-contained installer. A zip-based package may also be available as a fallback for users or mod managers that prefer extracted files.
